# 온라인 영화 예매 시스템 SRS (Software Requirements Specification)

## 1. 개요

### 1.1 목적

본 문서는 MSA(Micro Service Architecture) 기반의 온라인 영화 예매 시스템 구축을 위한 기능적/비기능적 요구사항을 정의한다.

시스템은 사용자가:

* 영화 조회
* 상영 일정 조회
* 좌석 선택
* 영화 예매
* 결제
* 예매 취소/환불

를 수행할 수 있도록 제공한다.

또한 대규모 트래픽 환경에서도:

* 고가용성
* 확장성
* 데이터 정합성
* 장애 복원성

을 보장해야 한다.

# 2. 시스템 범위

* 사용자 인증
